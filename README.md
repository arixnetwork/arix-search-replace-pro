== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "Search & Replace" in your admin sidebar

== Frequently Asked Questions ==

= Is this safe to use? =
Yes! The plugin includes a preview mode so you can see exactly what will change before applying. It also requires admin privileges and uses nonces for security.

= Does it work with WooCommerce? =
Yes! It fully supports WooCommerce products and will update product descriptions, titles, and meta content.

= Can I undo changes? =
WordPress revisions are preserved for posts/pages. For safety, always backup your site before large operations.
